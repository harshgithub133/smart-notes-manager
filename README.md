# Smart Notes Manager

📌 Project Description  
A simple Python program to add, view, and delete notes stored in a file.

🚀 Features  
- Add notes  
- View all notes  
- Delete notes  

🛠️ Technologies Used  
- Python  
- File Handling  

▶️ How to Run  
1. Make sure Python is installed.  
2. Run:  
   ```bash
   python smart_notes.py
